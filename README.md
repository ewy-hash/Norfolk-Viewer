# Norfolk-Viewer

**Norfolk nuisance flood viewer**

## Contributors: 

Evan Wyers and Adam Akins

## **Organization of Repository:**

Data/: Holds the larger data files downloaded from the internet and generated by Working-doc.Rmd

deployable data/: Holds data to be uploaded to shiny app.

Plots: Holds some intermediate screenshots of plots generated by working-doc.rmd

rsconnect: holds information for deployment of shiny app

working-doc.Rmd: Used to explore the data originally and to generate files for shiny app

shiny-app.R: This is the usable shiny app.
